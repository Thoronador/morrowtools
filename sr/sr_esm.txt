
Format of Skyrim's ESM PlugIn Files

(All that follows below is still an assumption!)


  Record
  ======
    4 bytes: char Name[4]
        4-byte record name string (not null-terminated)
    4 bytes: long Size
        Size of the record not including the 20 bytes of header data.
	16 bytes:
		Unknown values, probably four groups of 4-byte integers/ flags



  (Sub-)Record
  ============
    4 bytes: char Name[4]
        4-byte sub-record name string (not null-terminated)
    2 bytes: uint16 Size
        Size of the sub-record not including the 6 bytes of header data.
    ? bytes: Sub-Record data.
        Format depends on the sub-record type).


  Groups:
  =======
    GRUP
      4 byte - uint32 - size of group, including the header
      4 byte - char[4] - name of group to follow
      12 byte - unknown data
      ? bytes - group data


Records and their sub-records:

  TES4: Main header(?)
      HEDR (12 bytes)
        Unknown, possibly three 4-byte values?
        4 bytes - float, version number
        4 bytes - uint32_t, number of records and groups
        4 bytes - uint32_t, next object ID available
      CNAM (string, NUL-terminated)
        author name
      SNAM (string, NUL-terminated, optional)
        summary / description
      MAST (string, NUL-terminated, optional, can occur multiple times)
        master file name (dependency)
      DATA (8 bytes, optional, can occur multiple times)
        unknown data
        -> Always occurs after MAST. MAST and DATA form pairs that can occur
           multiple times.
      ONAM (size varies, but is an integral multiple of 4 bytes; optional)
        unknown list of form IDs
      INTV (4 bytes)
        unknown integer value


  AACT: action(?)
      EDID (string, NUL-terminated)
          editorID


  ACHR: character reference
      EDID (string, NUL-terminated, optional)
          editor ID
      VMAD (sizes varies, optional)
          unknown data
      NAME (4 bytes)
          uint32_t base object form ID
      XRGD (size varies (somewhere near 500 bytes), optional)
          unknown data
      XRGB (12 bytes, optional)
          unknown data
      XLCM (4 bytes, optional)
          unknown data
      XLKR (8 bytes or 4 bytes, can occur multiple times, optional)
          linked reference
            uint32_t keyword form ID (zero if none) (skipped in four byte version, I assume)
            uint32_t reference form ID
      INAM (4 bytes, optional)
          unknown data
      PDTO (8 bytes, optional)
          unknown data
      XAPD (1 bytes, optional, can occur multiple times)
          unknown data
      XAPR (8 bytes, optional, can occur multiple times)
          unknown data

          ----> XAPD/XAPR records seem to form pairs (most of the time).

      XLCN (4 bytes, optional)
          persistence location form ID (points to record type LCTN)
      XLRT (4 bytes, optional)
          location reference type form ID (points to record type LCRT)
      XEZN (4 bytes, optional)
          encounter zone form ID (points to record type ECZN)
      XOWN (4 bytes, optional)
          owner form ID (points to FACT or sometimes NPC_)
      XESP (8 bytes, optional)
          unknown data, looks like two uint32_ts
            uint32_t  Enable Parent Reference Form ID
            uint32_t  flags
                        0x01 - Set Enable State to opposite of parent
                        0x02 - Pop In
      XSCL (4 bytes, optional)
          float object scale
      .... (varies)
          some more unknown subrecords
      DATA (24 bytes)
          reference data


   ACTI: activator
      EDID (string, NUL-terminated)
          editorID
      VMAD (?? bytes, optional)
          unknown data
      OBND (12 bytes)
          unknown
      FULL (4 bytes, optional)
          unknown
      MODL (string, NUL-terminated, optional)
          model path
      MODT (?? bytes, optional)
          unknown
      MODS (?? bytes, optional)
          unknown
      DEST (8 bytes, optional)
          unknown

        DSTD (20 bytes, optional)
            unknown
        DMDL (string, NUL-terminated, optional)
            another model path
        DMDT (?? bytes, optional)
            unknown
        DMDS (?? bytes, optional)
            unknown
        DSTF (zero bytes, optional)
            unknown

        --> DSTD up to DSTF form a structure, DSTF finishes it.

      KSIZ (4 bytes, optional)
          number of uint32_ts in KWDA
      KWDA (multiple of 4 bytes), follows KSIZ, therefore optional
          array of formIDs/uint32_t2, number of elements is specified in KSIZ
      PNAM (4 bytes, optional)
          Default Primitive Colour
          uint8_t Red
          uint8_t Green
          uint8_t Blue
          uint8_t zero (seems to be always zero)
      SNAM (4 bytes, optional)
          uint32_t Looping Sound Form ID (points to SNDR)
      VNAM (4 bytes, optional)
          uint32_t Activate Sound Form ID (points to SNDR)
      WNAM (4 bytes, optional)
          uint32_t Water Type Form ID (points to WATR)
      RNAM (4 bytes, optional)
          uint32_t Activate Text Override string ID
      FNAM (2 bytes, optional)
          unknown
      KNAM (4 bytes, optional)
          uint32_t Interaction Keyword Form ID (points to KYWD)


  ADDN: AddOnNode
      EDID (string, NUL-terminated)
          editorID
      OBND (12 bytes)
          unknown data
      MODL (string, NUL-terminated)
          model path
      MODT (size varies)
          unknown data
      DATA (4 bytes)
          unknown data - contains addon node index as uint16(?)
      SNAM (4 bytes, optional)
          sound descriptor form ID (points to SNDR)
      DNAM (4 bytes)
          uint16_t Master Particle System Cap
          uint16_t flags
             0x01 - unknown (always set?)
             0x02 - "always loaded" flag


  ALCH: alchemy stuff (potions?)
      EDID (string, NUL-terminated)
          editorID
      OBND (12 bytes)
          unknown data
      FULL (4 bytes, optional)
          unknown data
      KSIZ (4 bytes, optional)
          number of items in KWDA
      KWDA (n * 4 bytes, optional)
          array of form IDs/uint32_ts
      MODL (string, NUL-terminated)
          model path
      MODT (?? bytes)
          unknown data
      MODT (?? bytes, optional)
          unknown data
      YNAM (4 bytes, optional)
          uint32_t Pickup Sound Form ID (points to SNDR)
      ZNAM (4 bytes, optional)
          uint32_t Putdown Sound Form ID (points to SNDR)
      ETYP (4 bytes, optional)
          uint32_t Equip Type Form ID (points to EQUP)
      DATA (4 bytes)
          float Weight
      ENIT (20 bytes)
          uint32_t Potion Value
          uint32_t Flags
              0x00000001 - Do NOT autocalculate
              0x00000002 - Food Item
              0x00010000 - Medicine
              0x00020000 - Poison
          uint32_t ??? (maybe Addiction form ID?)
          float    Addiction Chance (ranges from 0.0 to 1.0?)
          uint32_t Use Sound Form ID (points to SNDR)
      EFID (4 bytes)
          unknown data
      EFIT (12 bytes)
          unknown data, but first 4 bytes seem to be a float


  AMMO: ammunition
      EDID (string, NUL-terminated)
          editorID
      OBND (12 bytes)
          unknown
      FULL (4 bytes, optional)
          string ID of object's full name
      MODL (string, NUL-terminated, optional)
          model path
      MODT (?? bytes, optional)
          unknown data
      YNAM (4 bytes)
          uint32_t Pickup Sound form ID (points to SNDR)
      ZNAM (4 bytes)
          uint32_t Putdown Sound form ID (points to SNDR)
      DESC (4 bytes)
          uint32_t string ID of object description
      KSIZ (4 bytes, optional)
          number of keywords in following KWDA
      KWDA (n * 4 bytes, optional)
          array of form IDs of keywords
      DATA (16 bytes)
          uint32_t  form ID of PROJ record
          uint32_t  flags(?)
          float     base damage
          uint32_t  value


  ANIO: animated object
      EDID (string, NUL-terminated)
          editor ID
      MODL (string, NUL-terminated)
          model path
      MODT (size varies, optional)
          unknown data
      BNAM (string, NUL-terminated, optional)
          unknown data


  APPA: apparatus
      EDID (string, NUL-terminated)
          editorID
      OBND (12 bytes)
          unknown data
      FULL (4 bytes)
          unknown data
      QUAL (4 bytes)
          unknown data
      DESC (4 bytes)
          unknown data
      DATA (8 bytes)
          uint32_t value
          float    weight


  ARMO: armour
      EDID (string, NUL-terminated)
          editor ID
      VMAD (size varies, optional)
          unknown data
      OBND (12 bytes)
          unknown data
      FULL (4 bytes, optional)
          string ID of name
      EITM (4 bytes, optional)
          uint32_t Enchanting form ID (points to ENCH)
      MOD2 (string, NUL-terminated, optional)
          model path
      MO2T (size varies, optional)
          unknown data
      MO2S (size varies, optional)
          unknown data
      MOD4 (string, NUL-terminated, optional)
          model path
      MO4T (12 bytes, optional)
          unknown data
      MO4S (size varies, optional)
          unknown data
      BODT (12 bytes or 8 bytes)
          unknown data
      ETYP (4 bytes, optional)
          uint32_t Equip Type Form ID (points to EQUP)
      BIDS (4 bytes, optional)
          uint32_t Block Bash Impact Data Set Form ID (points to IPDS)
      BAMT (4 bytes, optional)
          uint32_t Alternate Block Material Form ID (points to MATT)
      YNAM (4 bytes, optional)
          uint32_t Pickup Sound Form ID (points to SNDR)
      ZNAM (4 bytes, optional)
          uint32_t Putdown Sound Form ID (points to SNDR)
      RNAM (4 bytes)
          unknown data
      KSIZ (4 bytes, optional)
          number of uint32_ts in KWDA
      KWDA (n * 4 bytes, optional)
          array of uint32_ts/form IDs
      DESC (4 bytes, optional)
          string ID of description (mostly zero, i.e. unused)
      MODL (4 bytes, can occur 1+ times)
          uint32_t Model Form ID (points to ARMA)
      DATA (8 bytes)
          uint32_t value
          float    weight
      DNAM (4 bytes)
          unknown data
      TNAM (4 bytes, optional)
          uint32_t Template Armor form ID (points to ARMO)


  ARTO: art object
      EDID (string, NUL-terminated)
          editor ID
      OBND (12 bytes)
          unknown data
      MODL (string, NUL-terminated)
          model path
      MODT (size varies, optional)
          unknown data
      DNAM (4 bytes)
          unknown data


  ASPC: acoustic space
      EDID (string, NUL-terminated)
          editor ID
      OBND (12 bytes)
          unknown data
      SNAM (4 bytes, optional)
          uint32_t looping sound form ID (points to SNDR (sound descriptor))
      RDAT (4 bytes, optional)
          uint32_t region form ID (points to REGN), "use sounds from region"
      BNAM (4 bytes, optional)
          uint32_t form ID of environment type (points to REVB)


  ASTP: association type
      EDID (string, NUL-terminated)
          editor ID
      MPRT (string, NUL-terminated)
          male parent
      FPRT (string, NUL-terminated)
          female parent
      MCHT (string, NUL-terminated, optional)
          male child
      FCHT (string, NUL-terminated, optional)
          female child
      DATA (4 bytes)
          uint32_t flags
              0x00000001 - Familiy Association


  BOOK: book
      EDID (string, NUL-terminated)
          editorID
      VMAD (?? bytes, optional)
          unknown data
      OBND (12 bytes)
          unknown
      FULL (4 bytes, optional)
          string ID of book's title
      MODL (string, NUL-terminated)
          model path
      MODT (size varies)
          unknown data
      DESC (4 bytes)
          string ID of the book's text
      YNAM (4 bytes, optional)
          uint32_t Pickup Sound form ID (points to SNDR)
      ZNAM (4 bytes, optional)
          uint32_t Putdown Sound form ID (points to SNDR)
      KSIZ (4 bytes, optional)
          number of uint32_ts in KWDA
      KWDA (multiple of 4 bytes), follows KSIZ, therefore optional
          array of formIDs/uint32_t2, number of elements is specified in KSIZ
      DATA (10 bytes)
          unknown data
      INAM (4 bytes, optional)
          uint32_t Inventory Art form ID (points to STAT)
      CNAM (4 bytes)
          unknown


  CAMS: Camera shot(?)
      EDID (string, NUL-terminated)
          editorID
      MODL (string, NUL-terminated, optional)
          model path
      MODT (12 bytes, optional)
          unknown data
      DATA (40 bytes or 44 bytes (later one since Skyrim 1.5.24))
          cam shot data (arrangement for 44 byte version)
            uint32_t    Camera Action
                          0x00 - Shoot
                          0x01 - Fly
                          0x02 - Hit
                          0x03 - Zoom
            uint32_t    Camera Location
                          0x00 - Attacker
                          0x01 - Projectile
                          0x02 - Target
                          0x03 - Lead Actor
            uint32_t    Camera Target
                          0x00 - Attacker
                          0x01 - Projectile
                          0x02 - Target
                          0x03 - Lead Actor
            uint32_t    Flags
                          0x01 - Position Follows Location
                          0x02 - Rotation Follows Target
                          0x04 - Don't Follow Bone
                          0x08 - First Person Camera
                          0x10 - No Tracer
                          0x20 - Start At Time Zero
            float       time multiplier: player (min.: 0.0f, max.: 20.0f)
            float       time multiplier: target (min.: 0.0f, max.: 20.0f)
            float       time multiplier: global (min.: 0.0f, max.: 1.0f)
            float       Max Time (min.: 0.0f, max.: 120.0f)
            float       Min Time (min.: 0.0f, max.: 120.0f)
            float       Target % Between Actors (min.: 0.0f, max.: 100.0f)
            float       Near Target Distance (min.: 0.0f, max.: 2000.0f)

              ----> The last float value (Near Target Distance) is presumably
                    the one missing in the 40 byte versions. In that case it
                    defaults to zero.
      MNAM (4 bytes)
          Imagespace Mod form ID (points to IMAD)


  CELL: cell
      EDID (string, NUL-terminated)
          editor ID
      FULL (4 bytes, optional)
          name string ID
      DATA (2 bytes, sometimes 1 byte)
          unknown data
      XCLC (12 bytes, optional)
          int32_t grid location X
          int32_t grid location Y
          uint32  unknown
      TVDT (684 bytes, optional)
          unknown data
      MHDT (1028 bytes, optional)
          unknown data
      XCLL (92 bytes)
          unknown data
      LTMP (4 bytes)
          lighting template form ID (points to LGTM)
      LNAM (4 bytes, optional)
          unknown data
      XCLW (4 bytes)
          unknown float(?)
      XCLR (size varies, optional)
          unknown data, but looks like array of form IDs
      XNAM (1 byte, optional)
          unknown data
      XLCN (4 bytes, optional)
          location form ID (points to LCTN)
      XWCN (4 bytes, optional)
          unknown data
      XWCS (4 bytes, optional)
          unknown data
      XWCU (64 bytes, optional)
          unknown data
      XCIM (4 bytes, optional)
          image space form ID (points to IMGS)
      XEZN (4 bytes, optional)
          form ID of encounter zone (points to ECZN)
      XCWT (4 bytes, optional)
          unknown data
      XCMO (4 bytes, optional)
          music type form ID (points to MUSC)
      XWEM (string, NUL-terminated, optional)
          unknown texture path
      XOWN (4 bytes, optional)
          owner faction form ID (points to FACT)
      XILL (4 bytes, optional)
          lock list form ID (points to FLST)
      XCCM (4 bytes, optional)
          region form ID
      XCAS (4 bytes, optional)
          default acoustic space form ID (points to ASPC)


  CLAS: classes
      EDID (string, NUL-terminated)
          editorID
      FULL (4 bytes)
          string ID of class name
      DESC (4 bytes)
          unknown data
      DATA (36 bytes)
          unknown data


  CLFM: colour ... (something)
      EDID (string, NUL-terminated)
          editor ID
      FULL (4 bytes, optional)
          string ID of name (matches editor ID in most cases)
      CNAM (4 bytes)
          unknown data
      FNAM (4 bytes)
          unknown data


  CLMT: climate
      EDID (string, NUL-terminated)
          editor ID
      WLST (integral multiple of 12 bytes, optional)
          uint32_t Weather Type form ID (points to WTHR)
          uint32_t Chance
          uint32_t Global form ID (points to GLOB)
      FNAM (string, NUL-terminated)
          path to Sun texture (.dds)
      GNAM (string, NUL-terminated)
          path to Sun Glare texture (.dds)
      MODL (string, NUL-terminated)
          model path
      MODT (12 bytes, optional)
          unknown data
      TNAM (6 bytes)
          unknown data


  COBJ: craftable object (recipes)
      EDID (string, NUL-terminated)
          editorID
      COCT (4 bytes) (not present, if there are no CNTO subrecords)
          component count (uint32_t)
      CNTO (8 bytes, occurs 0+ times)
          formID of required component (uint32_t)
          count (uint32_t)
      CTDA (32 bytes, can occur multiple times, optional)
          unknown
      CNAM (4 bytes)
          some formID
      BNAM (4 bytes)
          another formID
      NAM1 (2 bytes)
          resulting quantity (uint16_t)


  COLL: collision stuff
      EDID (string, NUL-terminated)
          editor ID
      DESC (4 bytes)
          description string ID(?)
      BNAM (4 bytes)
          unknown data
      FNAM (4 bytes)
          unknown data
      GNAM (4 bytes)
          unknown data
      MNAM (string, NUL-terminated)
          equal to editor ID in most (all?) cases
      INTV (4 bytes)
          number of form IDs in following CNAM record
      CNAM (n * 4 bytes, optional)
          array of form IDs; only present, if INTV > zero


  CONT: container
      EDID (string, NUL-terminated)
          editor ID
      VMAD (?? bytes, optional)
          unknown data
      OBND (12 bytes)
          unknown data
      FULL (4 bytes, optional)
          name string ID
      MODL (string, NUL-terminated)
          model path
      MODT (?? bytes)
          unknown data
      MODS (?? bytes, optional)
          unknown data
      COCT (4 bytes)
          component count, i.e. number of following CNTO subrecords
      CNTO (8 bytes, can occur multiple times)
          uint32_t form ID of object
          uint32_t object count
      COED (12 bytes, optional)
          unknown data
      DATA (5 bytes)
          uint8_t flags(?)
            0x02 - Respawns
            0x04 - Show Owner
          float   weight
      SNAM (4 bytes, optional)
          uint32_t Open Sound form ID (points to SNDR)
      QNAM (4 bytes, optional)
          uint32_t Close Sound form ID (points to SNDR)

      --> flags in record header:
          0x00010000 - Random Anim Start
          0x02000000 - Obstacle


  CPTH: camera path
      EDID (string, NUL-terminated)
          editor ID
      CTDA (20 bytes, can occur multiple times)
          unknown data
      CIS1 (string, NUL-terminated, optional)
          var name(?)
      ANAM (8 bytes)
          uint32_t parent camera path form ID (points to CPTH)
          uint32_t next camera path form ID (points to CPTH)
      DATA (1 byte)
          uint8_t flags and camera zoom settings
            0x00 - Camera Zoom: Default
            0x01 - Camera Zoom: Disable
            0x02 - Camera Zoom: Shot List
            0x80 - inverse(!) "Must have Camera Shots" flag
      SNAM (4 bytes, can occur multiple times)
          uint32_t camera shot form ID (points to CAMS)


  CSTY: combat style
      EDID (string, NUL-terminated)
          editor ID
      CSGD (40 bytes or 32 bytes or 8 bytes)
          float Offensive Mult
          float Defensive Mult
          float Group Offensive Mult
          float Melee (Equipment Score Mult)
          float Magic (Equipment Score Mult)
          float Ranged (Equipment Score Mult)
          float Shout (Equipment Score Mult)
          float Unarmed (Equipment Score Mult)
          float Staff (Equipment Score Mult)
          float Avoid Threat Chance

          --> Staff and Avoid Threat Chance are missing in 32-byte version,
              Staff is set to 1.0f there, ATC to 0.2f.

          --> Additionally, Group Offensive Mult, Melee, Magic, Ranged, Shout
              and Unarmed are missing in the 8-byte version, these values
              default to 1.0f there.

      CSMD (8 bytes, optional)
          unknown data, but these are two floats
      CSME (28 bytes or 32 bytes, optional)
          unknown data, looks like series of floats
      CSCR (16 bytes or 8 bytes, optional)
          float Circle Mult (Close Range, Dueling)
          float Fallback Mult (Close Range, Dueling)
          float Flank Distance (Close Range, Flanking)
          float Stalk Time (Close Range, Flanking)

          --> Last two seem to be missing in 8-byte version, they default
              to 0.2f in that case.
      CSLR (4 bytes, optional)
          float Long Range Strafe Mult

          --> Defaults to 0.2f, if not present.

      CSFL (20 bytes or 32 bytes, optional)
          unknown data, could be series of floats
      DATA (4 bytes, optional)
          unknown data, possibly flags


  DEBR: Debris
      EDID (string, NUL-terminated)
          editor ID
      DATA (size varies, can occur multiple times)
          uint8_t Percentage
          string  ModelFilename (NUL-terminated)
          uint8_t Flags
                    0x01 - Has Collision Data
      MODT (size varies, optional, can occur multiple times)
          unknown data

        --> DATA and MODT form pairs that can occur more than once. However,
            MODT is an optional component in those pairs.


  DIAL: dialogue record
      EDID (string, NUL-terminated, optional)
          editor ID
      FULL (4 bytes, optional)
          uint32_t string ID
      PNAM (4 bytes)
          unknown
      BNAM (4 bytes, optional)
          uint32_t dialogue branch form ID (points to DLBR)
      QNAM (4 bytes)
          unknown
      DATA (4 bytes)
          unknown
      SNAM (4 bytes)
          unknown, seems like char[4]
      TIFC (4 bytes)
          uint32_t counter


  DLBR: dialog branch(?)
      EDID (string, NUL-terminated)
          editorID
      QNAM (4 bytes)
          unknown data
      TNAM (4 bytes)
          unknown data
      DNAM (4 bytes)
          unknown data
      SNAM (4 bytes)
          unknown data


  DLVW: dialog view(?)
      EDID (string, NUL-terminated)
          editorID
      QNAM (4 bytes)
          unknown data
      TNAM (4 bytes, can occur multiple times, optional)
          unknown data
      BNAM (4 bytes, can occur multiple times, optional)
          unknown data

          ---> only TNAM or BNAM can be present in a DLVW record, not both at the same time

      ENAM (4 bytes, optional)
          unknown data
      DNAM (1 byte, optional)
          unknown data


  DOBJ: unknown record type
      DNAM (2424 bytes)
        array of 8-byte elements:
          char[4]  label
          uint32_t formID


  DOOR: door
      EDID (string, NUL-terminated)
          editorID
      VMAD (size varies, optional)
          unknown data
      OBND (12 bytes)
          unknown data
      FULL (4 bytes, optional)
          unknown data
      MODL (string, NUL-terminated, optional)
          model path
      MODT (36 bytes, optional)
          unknown data
      MODS (size varies, optional)
          unknown data
      SNAM (4 bytes, optional)
          unknown data
      ANAM (4 bytes, optional)
          unknown data
      FNAM (1 byte)
          unknown data


  DUAL: dual cast data
      EDID (string, NUL-terminated)
          editorID
      OBND (12 bytes)
          unknown data
      DATA (24 bytes)
          the dual cast data
            uint32_t  projectile form ID
            uint32_t  explosion form ID
            uint32_t  effect shader form ID
            uint32_t  hit effect art form ID (art object)
            uint32_t  impact data set form ID
            uint32_t  flags
                        0x01 - hit effect art inherits scale
                        0x02 - projectile inherits scale
                        0x04 - explosion inherits scale


  ECZN: encounter zone
      EDID (string, NUL-terminated)
          editor ID
      DATA (8 or 12 bytes)
          unknown data; 8 bytes occur twice, 12 bytes occur 278 times


  EFSH: Effect Shader
      EDID (string, NUL-terminated)
          editor ID
      ICON (string, NUL-terminated)
          texture path
      ICO2 (string, NUL-terminated)
          texture path
      NAM7 (string, NUL-terminated)
          texture path
      NAM8 (string, NUL-terminated, optional)
          texture path
      NAM9 (string, NUL-terminated, optional)
          texture path
      DATA (size varies, 308-400 bytes, always an integral multiple of four bytes)
          shader data


  ENCH: enchantments
      EDID (string, NUL-terminated)
          editorID
      OBND (12 bytes)
          unknown data
      FULL (4 bytes, optional)
          string ID of enchantment's name
      ENIT (36 bytes or 32 bytes in earlier versions)
          unknown data
      EFID (4 bytes, can occur multiple times)
          unknown data
      EFIT (12 bytes, can occur multiple times)
          unknown data
      CTDA (32 bytes, optional)
          unknown data
      CIS (string, NUL-terminated, optional)
          variable name - requires CTDA

          --> EFID and EFIT and CTDA/CIS2 seem to form blocks


  EQUP: equipment slot
      EDID: (string, NUL-terminated)
          editorID
      PNAM (n*4 bytes, optional)
          parent slots' form IDs
      DATA (4 bytes)
          uint32_t flags
            0x01 - Use All Parents


  EXPL: explosions
      EDID (string, NUL-terminated)
          editorID
      OBND (12 bytes)
          unknown data
      FULL (4 bytes, optional)
          name string ID
      MODL (string, NUL-terminated, optional)
          modelPath
      MODT (unknown size, optional)
          unknown data
      EITM (4 bytes, optional)
          enchantment form ID (record type ENCH)
      MNAM (4 bytes, optional)
          image space modifier form ID (record type IMAD)
      DATA (52 bytes or 48 bytes or 44 bytes or 40 bytes)
